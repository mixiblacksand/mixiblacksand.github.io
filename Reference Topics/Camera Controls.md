# Mouse Controls
```
Orbit : Drag MMB
Pan : Shift Drag MMB
Zoom : Ctrl Drag MMB or Scroll Mouse Wheel
Snap Orbit : Hold Alt while Dragging MMB
Quick Ortho View : Hold Alt and Press MMB while moving the mouse
```
Quick Ortho View is a 'flick' gesture that will rotate to the next orthographic view in the direction you moved the mouse.

# Keyboard Controls
```
Orbit : Numpad 2, 4, 6, 8
Pan : Ctrl + Numpad 2, 4, 6, or 8
Zoom : Numpad + or -
Roll : Shift + Numpad 4 or 6
Focus Selection : Numpad Period
Isolate Selection : Slash or Numpad Slash
Look Through Camera Object : Numpad 0
```
# Fly/Walk Mode (WASD Controls)
```
Toggle Fly Mode : Shift-`
Movement : WASD
Height : Q or E
Speed Adjust : Shift or Alt
Teleport : Space
Toggle Gravity : Tab
```
Note that the world grid is NOT solid. You need to be above a mesh before you toggle gravity!
