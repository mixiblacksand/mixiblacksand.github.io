Most of the time you'll be working from some kind of reference, either for a full character or for style, proportion, details, etc. You can add these images directly to Blender to refer to.
### Add an Image Editor Pane
1. Hover the mouse over the bottom border of the 3D viewport.
2. Right click and choose 'Vertical Split'.
3. Move the cursor to where you want the split to be and left click. You can click and drag the border at any time to tweak it.
4. In the upper left of the new panel, click the Editor Type menu and choose Image Editor.
5. In the top center of the Image Editor, click 'Open.'
6. Load an image, and it should appear in this pane.
7. Click the folder icon to load more images.
8. Use the dropdown next to the image name to switch images.

We can also place images directly in the scene to work over.
### Add a Reference Plane to the Scene
1. Press Shift-A in Object Mode in the 3D viewport.
2. Choose Image > Reference.
3. The reference image will appear facing the viewport camera. If this isn't the angle you want. you can move, rotate, and scale the plane.
