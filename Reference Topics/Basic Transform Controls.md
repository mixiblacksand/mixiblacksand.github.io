# Grab, Rotate, Scale
```
Grab : G
Rotate : R
Scale : S
```
After any of these commands, move the mouse. 

Left click to confirm. Right click to cancel.
# Lock Axis
Press X, Y, or Z during an action to lock to that axis.

Press Shift-X, Shift-Y, or Shift-Z to lock to a plane.

# Free Rotation
```
Free Rotation : R Twice
```
By default, Rotate works in the plane of the screen. Press R twice to rotate freely.

# Clear Transforms
```
Clear Location : Alt-G
Clear Rotation : Alt-R
Clear Scale : Alt-S
```
Pressing any of these will clear any changes made to its transforms.

# Apply Transforms
```
Apply Menu : Ctrl-A
```
To set an object's default state to a new one, use the Apply menu and choose the desired transform or combination of transforms.

>[!warning] Beware of Unwanted Object Transforms
>Transforms done at the object level can have significant effects on later operations. A common issue is objects with different scales or non-uniform scales causing unpredictable results. 
>
>If you are getting unexpected behavior when editing, you may need to apply your transforms.
