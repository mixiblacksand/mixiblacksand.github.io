What we can do in the 3D viewport depends what mode we are in.

# Object Mode
Object Mode is the default mode. We use this mode for:
* Adding and removing objects
* Transforming objects in [world space](VRC%20Character%20Book/Reference%20Topics/Global%20vs%20Local%20Space.md)
* Adding and applying [[modifiers]]
# Edit Mode
We can get to Edit Mode quickly by pressing `Tab`. Edit mode allows us to modify mesh data directly.
# Sculpt Mode
Sculpt mode allows us to edit meshes in a more organic way, pushing and pulling large numbers of points at once.
# Vertex Paint Mode
We can apply color data to vertices hre.
# Weight Paint Mode
Weight Paint mode allows us to adjust the value of vertex groups across the mesh. We'll primarily use this when skinning a mesh to a rig, but vertex groups have other uses across Blender.
# Texture Paint Mode
This is for painting textures onto a UV mapped mesh with a material that takes a texture.
