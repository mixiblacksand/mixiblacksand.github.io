The Snapping menu gives us lots of options about how we snap elements together. I'll just go over the ones we'll use the most.

![](Pasted%20image%2020260404002156.png)

# Face Project
This option projects a ray from the camera through the element, snapping it to any surface it hits. If you have a solid object selected in edit mode, this will flatten it against the surface, which is usually unwanted behavior.
# Face Nearest
With this on your selected elements will snap to the surface closest to them, regardless of the camera. This is useful if you have multiple elements selected and want them all to conform to a surface, a bit like the [[Shrinkwrap|Shrinkwrap modifier]]. 
# Volume
This is like face project, but instead of snapping to the surface, it snaps to the center of the volume it touches.
