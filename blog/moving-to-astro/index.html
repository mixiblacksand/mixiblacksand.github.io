I already jumped ship on 11ty

Tue Nov 18

Well that didn’t take long

A few days in and I’m already rebuilding the site on Astro. I like how it’s just kinda throwing Javascript I already know around instead of using another templating language.

Also nice that if you choose to write CSS in your layout files, it scopes them and links them properly in the head of the generated file. Trying to do the same thing in 11ty resulted in style tags all over the html unless you started passing variables back up to the base layout.

Auto image compression out of the box was very opinionated about having all images under the assets directory, but the playform/compress integration makes it work painlessly with the directory structure I prefer (media files in the folder with the post they’re for.)

I don’t strictly need the flexibility Astro offers right now, but as my needs grow I feel like Astro will meet them.
